# onetoeight.org

Marketing website for **one to eight** — eight essentials for a screen-born generation.

## Pages

- `/` — main landing page
- `/parents/fear` — "What does your teenager actually know?"
- `/parents/hope` — "A life screens can't compete with"
- `/parents/bridge` — "You didn't have a phone until you were 16"

## Stack

Static HTML. No build step. Just serve the files.

## Deploy

Pushed to Vercel automatically via GitHub. Domain `onetoeight.org` is the production target.
